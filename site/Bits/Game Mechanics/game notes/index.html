{ Default set var vG = true }

Ex:

if key_1 pressed, state = G{ If state = G{ Set var vG = true # this is for dialogue purposes and determining who is first in party } # Code for her stats ie var spd = 10; # etc } If key_2 pressed, state = N{ set var vN = true; # stats here }

Make sure all variables start with ‘v(char symbol)’ to differentiate it from the state. (Ie if G = true…)
